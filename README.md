# CredPal Take-Home - Pay Later Screen

## Overview
Flutter implementation of the "Pay Later" screen from the provided [Figma design](https://www.figma.com/file/KZKDiLKAnm04rPAKMx350o/Untitled?node-id=0%3A1&t=XHNQBxPrmi0NBVSu-1).

## Features Implemented
- Pixel-perfect UI matching Figma (header, search, horizontal products, featured merchants grid)
- Provider for state management (mock data)
- Responsive layout (tested on mobile & web)
- Custom colors from design
- Scrollaeble content

## Assumptions Made
- Images are placeholders
- Merchant avatars use colored circles with initials buy I used the figma logo as svg
- Font: Avenir
- Discount badge only shown when applicable

## Setup & Run
Make sure you already have [Flutter](https://flutter.dev) installed 

1. Clone repo: `git clone <your-repo>`
2. `flutter pub get`
3. Run:
   - Android: `flutter run`
   - Web: `flutter run -d chrome`
   - iOS: `flutter run` (on macOS) system only
